# 配置您的店面

AEM Assets集成显示在AEM Assets中管理的产品图像，而不是传统的Commerce托管的图像。 该集成提供了增强的图像管理功能：通过Adobe的Content Delivery Network (CDN)实现高级优化、裁剪和交付。

要将AEM Assets与由Edge Delivery Services提供支持的Commerce店面结合使用，请完成[EDS AEM Assets配置](https://experienceleague.adobe.com/developer/commerce/storefront/setup/configuration/aem-assets-configuration/)主题中所述的店面配置。
