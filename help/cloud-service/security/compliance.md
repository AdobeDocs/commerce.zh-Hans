# 法规遵从性和认证

[!DNL Adobe Commerce as a Cloud Service]建立在信任、透明和严格遵守全球安全和隐私标准的基础之上。 Adobe的企业级安全计划确保[!DNL Adobe Commerce as a Cloud Service]满足受监管行业和安全意识组织的期望。

## 行业认证

Adobe为支持[!DNL Adobe Commerce as a Cloud Service]的基础架构和服务维护了一整套认证和证明。 这些功能包括：

- **SOC 2 Type II**：独立审核验证Adobe对安全性、可用性和机密性的控制。

- **ISO/IEC 27001**： Adobe的信息安全管理系统(ISMS)已在此全球公认的标准下获得认证。

- **PCI DSS**： [!DNL Adobe Commerce]基础架构符合PCI DSS，确保安全处理付款数据。 客户有责任维护其自定义代码和扩展的PCI合规性。

- **GDPR和CCPA**： Adobe提供了一些工具和文档，以帮助客户履行数据隐私法规规定的义务。

有关Adobe的认证和合规性文档的完整列表，请访问[Adobe信任中心](https://www.adobe.com/trust/compliance/compliance-list.html){target="_blank"}。

## 从业者认证

Adobe还为使用[!DNL Adobe Commerce]的开发人员、架构师和商务从业人员提供了基于角色的认证。 这些认证可验证在安全实施和平台治理方面的专业知识。 有关详细信息，请参阅[Adobe认证门户](https://experienceleague.adobe.com/en/docs/certification/program/technical-certifications/ac/ac-overview){target="_blank"}。
